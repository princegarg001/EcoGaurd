🚀 Getting Started

Step 1: Get API Keys (20 minutes)

• Electricity Maps: https://www.electricitymap.org/

• GitLab Token: https://gitlab.com/-/profile/personal_access_tokens

Step 2: Set Environment Variables

export ELECTRICITY_MAPS_API_KEY="your-key" export GITLAB_TOKEN="your-token" export CI_PROJECT_ID="80410036"

Step 3: Run Data Collection

python collect_real_data.py

Step 4: View Dashboard

open dashboards/src/index.html
